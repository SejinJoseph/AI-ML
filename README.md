# 🏠 House Price Prediction using Linear Regression

## 📌 Overview
This project focuses on predicting house prices using **Linear Regression**, a supervised machine learning algorithm. The model is trained on housing data and predicts prices based on various input features.

---

## 🚀 Features
- Data preprocessing and cleaning
- Handling missing values
- Feature selection
- Model training using Linear Regression
- Model evaluation using R² score
- Predicting house prices from new input data

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 📊 Workflow
1. Data Collection  
2. Data Preprocessing  
3. Feature Engineering  
4. Model Training  
5. Model Evaluation  
6. Prediction  

---

## 📈 Model
The project uses **Linear Regression** to establish a relationship between independent variables (features) and the dependent variable (house price).
